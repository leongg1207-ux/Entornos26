# Informe Técnico

## 1. Descripción del problema

Durante la ejecución del programa se produce un error que provoca que la aplicación se detenga inesperadamente.  
El problema ocurre cuando el sistema intenta acceder a un recurso o variable que no está disponible o no ha sido inicializada correctamente.

Este fallo impide que el programa continúe su ejecución normal y afecta al funcionamiento general de la aplicación.

---

## 2. Cómo se reproduce

Pasos para reproducir el problema:

1. Ejecutar el programa desde la terminal.
2. Iniciar la función principal del sistema.
3. Introducir los parámetros o datos de entrada.
4. El programa intenta procesar los datos.
5. En ese momento se produce el error y la aplicación se detiene.
